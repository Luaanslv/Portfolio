Conhecimentos

Html

HTML é uma liguagem de marcação. Utilizamos para marcar elementos para definir as informações que a página vai exibir.

Css

CSS ou Cascading Style Sheet é responsavel pela estrutura estética. Foi criado para ser o estilo do HTML.

Javascript

Javascript também conhecido por js, é uma linguagem de programação interpretada estruturada com tipagem dinâmica.É ele que torna a aplicação interativa.

Bootstrap

É um framework que torna muitas etapas do desenvolvimento web rapido e dinâmicas. Ele é uma ferramenta de codigo aberto para desenvolvimento do html,css e js.
